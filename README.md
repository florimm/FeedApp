# FeedApp
## RSS Feed web application
build using:
    - dotnet core 2 preview
        - c#
        - javascript
        - DNTScheduler (for loading feeds from sources)

    - angular 4
        - ngrx-store
        - ngrx/effects
        - ngrx/router-store
        - ngrx/store-devtools

    - style
        - bootstrap
        - masonry
    - extra
        - server-side rendering
        - cache using ETAG

## running applications
'''
dotnet run
'''

## debug
